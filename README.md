NNTP2NNTP Proxy allows you to use your NNTP Account from multiple systems, each
with its own username and password. It fully supports SSL, and you can also limit
the access to the proxy with SSL certificates. NNTP2NNTP Proxy is very simple and
pretty fast.

This version has been updated to work with Python 3 and includes performance
improvements.

## Features

- Limit the number of connections for the server
- Limit the number of connections to the proxy per user
- Verification of client certificates can be disabled

## Installation

To install nntp2nntp, you can use pip:

```bash
pip install nntp2nntp
```

## Usage

For information on how to use nntp2nntp, please run the following command:

```bash
nntp2nntp.py
```
